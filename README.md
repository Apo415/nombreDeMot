# Projet

Exercice 3 : Nombre de mots.

## Description
Programme Java qui lit un fichier texte et compte
le nombre total de mots qu'il contient.

## Fonctionnement
- Lit chaque ligne du fichier avec Scanner + File
- Sépare chaque ligne par les espaces (split)
- Compte les mots ligne par ligne
- Affiche le total final

## Structure du projet
- `Main.java` : programme principal
- `expenses.txt` : fichier texte à analyser

## Exécution
1. Ouvrir le projet dans IntelliJ IDEA
2. Vérifier que expenses.txt est à la racine du projet
3. Lancer Main.java
